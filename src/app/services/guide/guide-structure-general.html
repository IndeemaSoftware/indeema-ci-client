Structure of Indeema CI - General


⚈ Projects - a list of Projects. Each projects could have many applcations. Each application has one server. Before cretaing projects setup templates, scripts and setup Server.

⚈ Servers - Instances of servers. To setup new server it's required to have ssh access wth sudo without password. Before creating server create templates and scripts in settings.

⚈ Settings - this is a starting page for new user. Create Templates and Scripts here or just install module with set of ready solutions.

⚈ Help - Indeema Ci doumentation and bug reporting.
