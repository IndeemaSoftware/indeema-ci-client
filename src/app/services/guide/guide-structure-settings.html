Structure of Indeema CI - Settings

⚈ Maintenance - list of html maintenance pages that can be used in service scripts when setuping application.

⚈ CI temapltes - list of ci scripts that can be used in service scripts when setuping application.

⚈ Server Dependencies - list of server package managers depending packages like apt-get or yum. These dependencies have also pre and post install scripts for better setup process.

⚈ Custom Dependencies - list of independant dependencies like npm or pip. Each dependancy is just a custom script to setup specific package.

⚈ Platforms - list of scripts for server setup. These scripts implement dirrect setup all all dependencies, firewalls and others.

⚈ Services - list of scripts for applications and projects setup. These scripts should configure certificates, paths and web servers.

⚈ Modules - on thsi page you can import and export your existing templates and scripts to share it with community. Also these is a list of modules that could be installed with one click.
